# Digital Wallet System (C++)

A console-based C++ digital wallet system simulating basic financial services. The project demonstrates object-oriented programming concepts along with file handling and data validation to manage user accounts and transactions.

---

## 🚀 Features

### User Panel
- User registration with CNIC, phone, and personal details  
- Secure login using CNIC and PIN  
- Deposit and withdraw funds  
- Transfer money between users  
- View transaction history  
- Persistent data storage using files  

### Admin Panel
- View all registered users  
- Monitor user balances  

---

## 🧠 Concepts Used

- Object-Oriented Programming (OOP)  
- File Handling (fstream)  
- Vectors (STL)  
- Input validation using regex  
- Basic system design simulation  

---

## 🎯 Learning Outcomes

- Understanding real-world banking system workflow  
- Implementation of user authentication system  
- Handling persistent data using file storage  
- Designing modular and reusable C++ classes  
- Managing transactions and balances in a structured system  

---

## ⚙️ How to Run

1. Open `main.cpp` in any C++ IDE (Visual Studio / VS Code / Code::Blocks)  
2. Compile and run the program  
3. Choose an option from the main menu  
4. Register or login to access wallet features  
5. Use PIN for authentication and manage transactions  

---

## 📌 Project Overview

This project simulates a simplified digital wallet system similar to real-world mobile banking applications. It allows users to manage accounts, perform transactions, and store data persistently using file handling.
